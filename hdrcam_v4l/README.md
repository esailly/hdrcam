# hdrcam_v4l

## Description

Capture webcam frames with varying exposure times, to compute HDR images, using 
[video4linux](https://www.linuxtv.org/). 

## Dependencies

- g++, cmake, pkg-config
- libv4l, v4l-utils
- opencv 3
